# clinical-qc-dashboard
Clinical Laboratory Quality Control Dashboard using Python (Levey–Jennings, Westgard rules, anomaly detection)
## Clinical Laboratory Quality Control Dashboard

### Overview
A data science project simulating clinical laboratory QC data to monitor
instrument performance using Levey–Jennings charts, Westgard rules,
and anomaly detection.

### Features
- Synthetic QC data generation
- Levey–Jennings visualization
- Westgard rule violations
- Instrument drift detection
- Machine learning–based anomaly detection

### Tech Stack
- Python
- pandas, numpy
- plotly
- scikit-learn

### Why This Project
Designed to demonstrate applied data science skills combined with
real-world clinical laboratory domain expertise.
